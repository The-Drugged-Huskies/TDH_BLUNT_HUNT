# BLUNT HUNT

**BLUNT HUNT** is a crypto-integrated arcade game on the **Dogechain** blockchain.

## Overview
Inspired by the classic *Duck Hunt*, **BLUNT HUNT** flips the script: instead of ducks, you are hunting for **blunts**!

## Gameplay
- **Objective**: Catch blunts shot into the air.
- **Weapon**: You start with a **Sling** as your default weapon.
- **Crypto Integration**: Players can upgrade their gear (update from the default sling) by paying wth crypto.
- **Setting**: The game begins on the first map, setting the scene for your blunt-hunting adventure.

## Tech Stack
- **Backend**: Python (Flask)
- **Blockchain**: Dogechain
